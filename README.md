## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Subodh-1370/Portfolio.git
```

### Navigate to Project

```bash
cd Portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
